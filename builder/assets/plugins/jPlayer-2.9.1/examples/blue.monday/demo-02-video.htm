jPlayer是完全自由和开放源码（MIT）用JavaScript编写的媒体库。一个jQuery的插件，（而现在的Zepto插件，） jPlayer可以让你快速编织跨平台的音频和视频到您的网页。jPlayer的全面的API允许你创建创新媒体解决方案。

引入相关资源

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
<script type="text/javascript" src="/jPlayer/js/jquery.jplayer.min.js"></script>

Jplayer.swf -> 不支持HTML5 <video> 和 <audio>的浏览器中使用了这个文件。

jPlayer.swf文件必须上传到你的服务器上。默认的位置是在一个相对当前页面名为"js"的文件夹中。 要修改这个flash文件的位置要使用构造函数配置swfPath。 下面是如何设置相对服务器根目录的swfPath路径。

$("#jpId").jPlayer( {
 swfPath:"/scripts/jPlayer"
});

jPlayerPlaylist获得播放列表相关的插件帮助。引入资源

<script type="text/javascript" src="../../dist/add-on/jplayer.playlist.min.js"></script>

js代码

$(document).ready(function(){
 new jPlayerPlaylist({// 播放列表
  jPlayer: "#jquery_jplayer_1",
  cssSelectorAncestor:"#jp_container_1"
 }, [
  {
   title:"Big Buck Bunny Trailer",
   artist:"Blender Foundation",
   free:true,
   m4v: "http://www.jplayer.org/video/m4v/Big_Buck_Bunny_Trailer.m4v",
   ogv: "http://www.jplayer.org/video/ogv/Big_Buck_Bunny_Trailer.ogv",
   webmv: "http://www.jplayer.org/video/webm/Big_Buck_Bunny_Trailer.webm",
   poster:"http://www.jplayer.org/video/poster/Big_Buck_Bunny_Trailer_480x270.png"
  }
 ], {
  swfPath: "../../dist/jplayer",// flash路径
  supplied: "webmv, ogv, m4v",// 视频格式
  useStateClassSkin: true,
  autoBlur: false,
  smoothPlayBar: false,
  keyEnabled: true,// 键盘控制
  autohide: {
   full: true // 播放时隐藏
  },
  size: { // 播放器宽高
   width: '100%',
   cssClass: 'jj' //自定义类名(css样式) ： 增加到.jp-video-270p上面
  }
 });
});
